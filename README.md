# Nuclear Renaissance Index

A self-contained research dashboard for nuclear energy equities — constituents,
price/volume history, technical indicators, an interactive optimizer, catalysts,
and a pre-IPO watchlist.

Live site: https://YOUR-USERNAME.github.io/nri/

## Update workflow

1. On your Mac, run `refresh.command` (from the `nuclear-index` folder) to pull
   fresh prices and regenerate `Nuclear-Renaissance-Index.html`.
2. Copy the fresh HTML into this repo as `index.html` (overwriting).
3. Commit + push (or drag-and-drop in the GitHub web UI). Pages re-deploys in
   about a minute.

Everything is a single HTML file — no build step, no dependencies, no server.
